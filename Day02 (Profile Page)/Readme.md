# Profile Page

A simple profile page built using HTML and CSS.

## Features

* Responsive design
* Navigation bar with links to different sections of the page
* Profile section with image, name, and description
* Skills section with list of skills
* Projects section with list of projects
* Section with links to social media and contact info

## Deployed Link

Github Pages Link: https://gauravprajapati03.github.io/Profile-Page/

## How to Access

To access this website, follow these steps:

1. Open a web browser on your device (e.g. Google Chrome, Mozilla Firefox, etc.)
2. Type the URL `https://gauravprajapati03.github.io/Profile-Page/` in the address bar and press Enter.
3. The website should load and display the profile page.
